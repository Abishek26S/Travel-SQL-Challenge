# 🌍 Travel SQL Challenge

Welcome to the Travel SQL Challenge!

This project is a hands-on SQL learning experience inspired by real-world travel analytics. It combines official tourism datasets with realistic business scenarios to help learners progress from SQL fundamentals to advanced analytical queries.

## Objectives

- Learn SQL through practical business problems
- Work with real and realistic travel datasets
- Practice joins, aggregations, CTEs, and window functions
- Build a portfolio-ready SQL project

## Planned Modules

- Customer Analytics
- Hotel Analytics
- Flight Analytics
- Booking Analytics
- Payment Analytics
- Reviews & Ratings
- Business Insights
- Executive Dashboard

## SQL Topics

- SELECT
- WHERE
- GROUP BY
- HAVING
- JOINs
- CASE
- Subqueries
- CTEs
- Window Functions
- Ranking Functions
- Date Functions

More content will be added as the project progresses.
